# Equation Randomizer

A fun and interactive equation randomizer game built with vanilla JavaScript. Test your math skills by solving randomly generated equations!

## Features

- Random equations with addition, subtraction, multiplication, and division
- Instant feedback on answers
- Score tracking
- Beautiful gradient UI
- Responsive design

## How to Use

1. Open `index.html` in your web browser
2. Solve the displayed equation
3. Enter your answer and press "Check" or Enter
4. Click "Next Equation" to get a new problem
5. Watch your score improve!

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
